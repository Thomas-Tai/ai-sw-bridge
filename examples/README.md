# Examples

Worked examples for ai-sw-bridge. Each subfolder is a complete, self-contained workflow you can run end-to-end.

| Example | What it demonstrates |
|---|---|
| [`minimal_cylinder/`](minimal_cylinder/) | Path C: record a cylinder once, parameterize, replay. Validates installation end-to-end. |

## Running an example

1. `cd` into the example folder
2. Follow the instructions in that folder's `README.md`

Each example ships a `locals.txt` and a `spec.json` but expects you to record your own `.swp` (since recordings are machine- and version-specific — see [docs/known_gotchas.md](../docs/known_gotchas.md)).
